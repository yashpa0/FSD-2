# React Login Form (MUI)

This is a simple React login form built using **Material UI (MUI)**.  
It includes basic validation and a clean, centered layout.

## Features
- Email and password fields
- Form validation
- Remember me checkbox
- Login type selection (User / Admin)
- Material UI components
- Responsive and clean design

## Technologies Used
- React
- Material UI (MUI)
- React Hooks

## Installation
1. Clone the repository
2. Install dependencies
   ```bash
   npm install


Learning Outcomes

After completing this project, the learner will be able to:

Create forms using React and Material UI components.
